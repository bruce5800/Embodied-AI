1. 安装yolo ，打开命令行cmd

执行    pip install ultralytics -i https://pypi.tuna.tsinghua.edu.cn/simple
         
           pip install ultralytics -i  https://mirrors.aliyun.com/pypi/simple

           pip install ultralytics -i  https://pypi.douban.com/simple

显示一个success字样。

2. 验证yolo正确的安装
 cmd 命令行执行  yolo -version
显示出来 8.xxx的版本 

常见错误： yolo找不到，电脑安全级别问题，手动配置环境变量。


3. yolo工具的初体验
cmd 命令行 执行
yolo predict model=yolov8n.pt  source='图片路径’
注意：yolov8n.pt pt文件必须在执行命令行的文件目录下。
          图片可以是本地的，也可以是在线的。

   yolo predict model=yolov8n.pt  source="0" show=True conf=0.4   # 推理0号摄像头， 显示预览画面。 置信度是0.4
